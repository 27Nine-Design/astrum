Navigation

Primary .nav--full-width

This is the primary navigation element that appears on the homepage using the modifier class .nav--full-width.

Example
  • Home
  • Our Work
  • What We Do
  • Who We Are
  • Our Journal
  • Get In Touch
Code 
<nav class="nav nav--primary nav--full-width">
    <ul class="nav__items">
        <li class="nav__item active"><a href="#">Home</a></li>
        <li class="nav__item"><a href="#">Our Work</a></li>
        <li class="nav__item"><a href="#">What We Do</a></li>
        <li class="nav__item"><a href="#">Who We Are</a></li>
        <li class="nav__item"><a href="#">Our Journal</a></li>
        <li class="nav__item"><a href="#">Get In Touch</a></li>
    </ul>
</nav>

Primary .nav--top-right

This is the primary navigation element positioned in the top right of the page using the modifier class .nav--top-right.

For pages where the navigation sits above a dark background the modifier class .nav--inverted can also be added to invert the text color.

Example
  • Home
  • Our Work
  • What We Do
  • Who We Are
  • Our Journal
  • Get In Touch
Code 
<nav class="nav nav--primary nav--top-right">
    <ul class="nav__items">
        <li class="nav__item active"><a href="#">Home</a></li>
        <li class="nav__item"><a href="#">Our Work</a></li>
        <li class="nav__item"><a href="#">What We Do</a></li>
        <li class="nav__item"><a href="#">Who We Are</a></li>
        <li class="nav__item"><a href="#">Our Journal</a></li>
        <li class="nav__item"><a href="#">Get In Touch</a></li>
    </ul>
</nav>

Secondary

This is the secondary navigation element used in the footer of the site. A simple listing of navigation items.

Example
  • Who We Are
  • What We Do
  • Our Journal
  • Get In Touch
Code 
<nav class="nav nav--secondary">
    <ul class="nav__items">
        <li class="nav__item"><a href="#">Who We Are</a></li>
        <li class="nav__item"><a href="#">What We Do</a></li>
        <li class="nav__item"><a href="#">Our Journal</a></li>
        <li class="nav__item"><a href="#">Get In Touch</a></li>
    </ul>
</nav>
